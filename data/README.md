# Data Layout

- `data/videos/` — подготовленные локальные видео после распаковки материалов из `lera_materials/`
- `data/artifacts/` — JSON-артефакты модулей `asr`, `ocr`, `det`
- `data/qdrant/` — локальные индексы `Qdrant`

Эти каталоги не коммитятся. Исходные материалы остаются в `lera_materials/`.
